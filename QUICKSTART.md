# 快速开始指南

## 第一步：安装FFmpeg

### Windows用户

1. 访问 [FFmpeg官网](https://ffmpeg.org/download.html#build-windows)
2. 下载Windows版本的FFmpeg
3. 解压到某个目录，例如 `C:\ffmpeg`
4. 将 `C:\ffmpeg\bin` 添加到系统PATH环境变量：
   - 右键点击"此电脑" -> "属性"
   - 点击"高级系统设置"
   - 点击"环境变量"
   - 在"系统变量"中找到"Path"，点击"编辑"
   - 点击"新建"，添加 `C:\ffmpeg\bin`
   - 点击"确定"保存

5. 验证安装：打开命令提示符，运行 `ffmpeg -version`

### macOS用户

```bash
brew install ffmpeg
```

### Linux用户

```bash
sudo apt update
sudo apt install ffmpeg
```

## 第二步：安装Python依赖

```bash
cd 重写翻译
pip install -r requirements.txt
```

## 第三步：启动应用

```bash
python main.py
```

## 使用流程

1. **打开视频**
   - 点击菜单栏"文件" -> "打开视频"
   - 或点击工具栏的"打开视频"按钮
   - 选择一个英文视频文件（支持MP4、MOV、MKV等格式）

2. **等待处理**
   - 系统会自动进行语音识别（使用Whisper）
   - 自动翻译字幕（使用Google Translate）
   - 进度条会显示处理进度
   - 处理时间取决于视频长度和计算机性能

3. **播放视频**
   - 处理完成后，视频会自动加载
   - 点击"播放"按钮开始观看

4. **查看字幕**
   - 右侧面板显示双语字幕
   - 当前播放的字幕会高亮显示
   - 自动滚动跟随播放进度

5. **跳转播放**
   - 点击任意字幕，视频会跳转到对应时间点
   - 方便快速定位到想要学习的句子

6. **使用复读功能**
   - 在字幕面板点击想要复读的句子
   - 在右侧控制面板选择复读次数（3/10/20次）
   - 点击"开始复读"按钮
   - 视频会循环播放该句子指定次数
   - 复读进度会实时显示

## 常见使用场景

### 场景1：学习英语口语
1. 打开英文视频
2. 观看时遇到不懂的句子
3. 点击该句子字幕
4. 选择复读10次
5. 跟随视频反复练习

### 场景2：听力训练
1. 打开英文视频
2. 先不看字幕盲听
3. 遇到听不懂的部分
4. 点击字幕查看翻译
5. 反复听直到听懂

### 场景3：记录笔记
1. 打开教学视频
2. 找到重要知识点
3. 点击对应字幕
4. 复读并做笔记
5. 继续观看

## 性能优化建议

- 使用较小的Whisper模型（如tiny或base）可加快处理速度
- 首次运行会下载Whisper模型，请耐心等待
- 处理后的字幕会缓存，再次打开同一视频会更快
- 如果翻译速度慢，可以在`config.py`中调整批次大小

## 注意事项

- 确保视频有清晰的英文音频
- 视频时长建议不超过1小时（处理时间较长）
- 需要稳定的网络连接（用于Google翻译）
- 处理大文件时需要足够的磁盘空间
